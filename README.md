# Mid-Project

#Zomato Bengaluru Restaurants: A
#Data-Driven Analysis
Mid Project After Cleansing 3 Epsilon AI
This presentation offers an in-depth analysis of the Zomato Bengaluru restaurant
dataset, providing key insights into the city's vibrant culinary landscape. Utilizing 20
distinct visualizations, we explore restaurant distribution, popular cuisines, customer
preferences, and operational characteristics that define Bengaluru's food scene. This
mid-project analysis, part of the Epsilon AI course, aims to deliver actionable
intelligence for restaurateurs, marketers, and food enthusiasts.
